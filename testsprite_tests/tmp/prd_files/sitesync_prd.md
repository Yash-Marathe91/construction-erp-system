# Sitesync Construction Platform PRD

## 1. Overview
Sitesync is a comprehensive construction management platform designed to streamline on-site operations, tracking attendance, inventory, and financial reporting.

## 2. Core Modules
### 2.1 Operations & Workforce
*   **Attendance**: Real-time check-in and check-out for construction labor and staff.
*   **Shift Management**: End-of-shift processing with operational verification.
*   **Payroll**: Configuration of base salary, PF, medical funds, and allowances.
*   **Employee Management**: View detailed roster with compact listing and advanced sorting.

### 2.2 Inventory & Materials
*   **Stock Tracking**: Real-time tracking of material levels (e.g., Cement, Steel).
*   **Catalog Management**: Adding new materials with unit specifications.
*   **Stock Adjustment**: Incremental or decremental stock updates with reasoning.

### 2.3 Finance & Reporting
*   **Invoicing**: Multi-line item invoice generation for vendors and site expenses.
*   **Audit Reports**: Detailed transaction logs with date filtering and PDF export.
*   **Receipts**: Interactive verification receipts for all disbursements.

### 2.4 AI Intelligence
*   **Sitesync AI**: A floating assistant that provides context-aware insights into site data.

## 3. Technical Requirements
*   **Frontend**: Next.js 16 (App Router), React 19.
*   **Backend**: Supabase Auth and Database.
*   **Styling**: Tailwind CSS (Tectonic Blueprint theme).
*   **Interactions**: Lucide icons, responsive modals, and high-density tables.

## 4. User Personas
*   **Storekeeper**: Manages inventory and material check-ins.
*   **Site Supervisor**: Manages daily attendance and shift closing.
*   **Admin**: Manages payroll, global settings, and user access.
